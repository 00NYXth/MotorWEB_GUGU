# 🌐 WinSearch – Simple Web Browser (WebView2)

**WinSearch** este un browser web simplu realizat în **C# Windows Forms**, care utilizează motorul **Microsoft Edge Chromium prin WebView2**.  
Aplicația permite navigarea pe internet, efectuarea de căutări Google și gestionarea mai multor tab-uri într-o interfață modernă.

---

## 📌 Caracteristici principale

- Căutare rapidă folosind **Google**
- Navigare web cu **Microsoft WebView2 (Chromium Engine)**
- Suport pentru **multiple tab-uri**
- Navigare **Back / Forward**
- **Refresh / Stop** încărcare pagină
- **Home Page** personalizat
- Interfață modernă inspirată de browser-ele actuale
- Deschiderea link-urilor `target="_blank"` în **tab nou**
